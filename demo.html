What is Machine Learning?

Machine learning is a subset of artificial intelligence that focuses on enabling systems to learn and improve from experience without being explicitly programmed. The core idea is that machines can identify patterns in data and make decisions with minimal human intervention.

One of the most important concepts in machine learning is the neural network, which is inspired by the structure of the human brain. These networks consist of interconnected nodes (neurons) that process information in layers, allowing the system to learn complex patterns and relationships in data.

Understanding Neural Networks

A neural network learns through a process called backpropagation. This is an algorithm that calculates the gradient of the loss function with respect to the network's weights, allowing the network to adjust and improve its predictions.

The training process involves feeding data through the network, comparing the output to the desired result, and then adjusting the weights backwards through the layers. This iterative process continues until the network achieves satisfactory performance on the training data.

  • Privacy First - No data sent to external servers

  • Contextual - Includes page title, URL, and section heading
  • Lightweight - Only ~3KB minified
  • Mobile Friendly - Works on all devices

How It Works

When you highlight text on a page using hilite2ai, the script automatically:

  1. Captures the selected text
  2. Finds the nearest preceding heading for context
  3. Grabs the page title and URL
  4. Formats everything into a structured prompt
  5. Copies it to your clipboard

Then you just paste it into your favorite AI assistant, and it has all the context it needs to give you a helpful, relevant explanation!

Integration Example

Adding hilite2ai to your website is simple:

<!-- Just add this script tag -->
<script src="https://cdn.jsdelivr.net/gh/yourusername/hilite2ai@1.0.0/hilite2ai.js"></script>

<!-- Optional: Show a notice banner -->
<div id="hilite2ai"></div>

That's it! Your entire website is now AI-enhanced.
