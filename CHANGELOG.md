# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-19

### Added
- Initial release of hilite2ai
- Text highlighting functionality
- Contextual prompt generation with:
  - Page URL
  - Page title
  - Nearest section heading
  - Selected text
- Copy to clipboard functionality
- Optional notification banner
- Mac/Windows keyboard shortcut detection (Cmd/Ctrl)
- Mobile responsive design
- Automatic selection clearing after copy
- Hover effects and smooth animations
- Ctrl+C/Cmd+C keyboard support to dismiss button
- Zero-dependency implementation
- MIT License

### Technical Details
- Pure vanilla JavaScript (ES6+)
- Self-contained IIFE pattern
- Inline CSS injection
- ~3KB file size
- Browser support: Chrome 90+, Firefox 88+, Safari 14+
